📊 Desafio Power BI – Replicação e Criação de Relatório Analítico
Este repositório contém a entrega do desafio proposto no curso da DIO – Formação Power BI Analyst.
O objetivo é replicar duas páginas já desenvolvidas durante o curso e criar uma terceira página original, aplicando os conceitos aprendidos sobre construção de visuais, organização de layout e publicação de relatórios.

🎯 Objetivo do Desafio
O desafio consiste em:

Replicar duas páginas do relatório criadas durante o curso utilizando a sample disponibilizada.

Criar uma terceira página original, contendo:

🌍 Mapa 1: Soma de Sales e Units Sold por país

🌎 Mapa 2: Soma de Profit por país

🥧 Gráfico de Pizza: Lucro por segmento
